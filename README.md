# Discord Bot
Discord Bot made with JavaScript

Discord JS V13

## setup
to setup create a 'config.json' file and add you TOKEN, clientId, and all the guildIds in a list
then run 'node ./deploy-commands.js' then 'node ./index.js'

#### stucture for config:

{

"token": "",

"clientId": "",

"guildId": ["", "", "", ""]

}
